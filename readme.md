# SSMApplicationTestByJenkins

## 项目目的
    1.完成框架课程作业
    2.测试Jenkins持续集成技术
    3.学习基本Linux操作
    4.练习Git代码托管
    5.练习Maven项目基本流程
    
## 项目使用技术
    1.Spring + SpringMVC + MyBatis框架整合
    2.Maven 的包管理器
    3.Jenkins 持续集成

## 项目支持的功能
    * 展示近一个月的收入与支出折线图
    * 提供结账功能
    * 提供水产品导入与导出的功能
    * 支持批量导入水产品信息
    * 支持导出水产品信息
    * 对操作进行记录，并提供日志帮助
    * 提供备份系统，避免出现奇怪的原因造成损失
        此功能只提供给管理员用户（高权限用户）
    * 提供简略搜索
